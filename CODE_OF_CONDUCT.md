# Code of Conduct

## Our Pledge

我们希望 `codex-desktop-orchestrator` 社区对所有参与者都是开放、友好、尊重的环境。

无论你的经验、身份、表达方式或背景如何，我们都欢迎你以建设性的方式参与讨论、提 issue、提交 PR 和分享经验。

## Our Standards

有助于建设社区的行为包括：

- 尊重不同背景和经验的参与者
- 对问题、设计和实现给出具体、善意的反馈
- 在讨论中尽量基于事实、复现步骤和代码证据
- 对新贡献者保持耐心，帮助他人理解上下文

不可接受的行为包括：

- 人身攻击、羞辱、嘲讽或骚扰
- 带有歧视、侮辱或敌意的表达
- 故意引战、刷屏、恶意消耗维护者时间
- 公开他人的私密信息、密钥或敏感日志

## Enforcement Responsibilities

项目维护者有责任澄清社区行为标准，并对不当行为采取适当且公平的处理措施。

维护者可以删除、编辑或拒绝不符合本规范的评论、issue、PR 或其他贡献内容。

## Scope

本规范适用于项目仓库内的所有互动，包括但不限于：

- Issue
- Pull Request
- 代码 review
- Discussions
- 与项目直接相关的社区交流

## Reporting

如果你遇到违反本规范的行为，请通过仓库 issue 或维护者提供的联系方式告知。

在处理相关问题时，维护者会尽量保护报告者的隐私。

## Attribution

本规范参考了 Contributor Covenant 的社区治理思路，并根据本项目的协作方式做了简化调整。
